# 연산자
## 연산자 구분
1. 단항 연산자
```
+x
-x
!x
```

2. 이항 연산자
```
x = y
x < y
x != y
```

3. 삼항 연산자
```
조건식:true?false
```


- - - -
### 대입 연산자

=	: 대입연산자

- - - -


### 산술연산자

+, -, *, /, %


- - - -

### 복합 대입연산자

+= , -=, *=, /=, %=


- - - -

### 관계 연산자

>, <, >=, <=, ==, !=


- - - -

### 증감 연산자

- - - -

### 논리 연산자

&&(AND), ||(OR), !(NOT)


- - - -

### 삼항 연산자

조건식? 식1:식2


- - - -

### 비트 연산자

&	(AND 연산)	a & b : a와 b가 모두 1이면 1
|	(OR 연산)	a | b : a와 b중 하나라도 1이면 1
^	(XOR 연산)	a ^ b가 같지 않으면 1
