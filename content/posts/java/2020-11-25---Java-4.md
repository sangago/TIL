# 특수문자와 서식문자

## 특수문자
\t	: 탭
\n	: 개행
\'	: 작은따옴표
\"	: 큰따옴표
\\	: 역슬래쉬


- - - -

## 서식문자
**printf() 메서드를 이용한다**

%d		10진수
%o		8진수
%x		16진수
%c		문자
%s		문자열
%f		실수

* 사용예시
```
//정수
System.out.printf("당신의 키는 %n cm입니다.", 170)
//문자
System.out.printf("제 혈액형은 %c 형입니다.", 'O')
//문자열
System.out.printf("%s 님, 제 이름은 %s입니다", "고길동","둘리")
```



- - - -

## 문자정렬

```
//왼쪽에서부터 정렬
System.out.println("d%", 123);		=> 123

//자릿수를 지정하여 오른쪽부터 정렬
System.out.println("5d%", 123);		=>   123

//소수점 자릿수를 지정하여 정렬
System.out.println(%.3, 1.23);		=> 1.230
```
