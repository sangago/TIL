# 배열
: 같은 자료형인 다수의 데이터를 인덱스를 이용하여 관리


* 배열 선언 후 초기화
```
int[] arr1 = new int[5];
arr1[0] = 100;
arr1[1] = 200;
arr1[2] = 300;
arr1[3] = 400;
arr1[4] = 500;
```


* 동시에 배열 선언과 초기화
```
int[] arr2 = {10, 20, 30, 40, 50};
```
