# 객체지향프로그래밍(OOP)

객체지향프로그래밍은 프로그래밍 기법 중 하나이다.


## 객체란?
: 세상에 존재하는 모든 것을 뜻하며, 프로그래밍에서 속성과 기능을 가지는 프로그램 단위

ex )
- 날씨 프로그램
속성 : 온도, 미세먼지
기능 : 날씨예보
- 사칙연산 프로그램
속성 : +, -, *, /
기능 : 연산기능

## 클래스란?
: 객체를 생성하기 위한 툴로 모든 객체는 클래스로부터 생성된다.




## 클래스의 구성요소
클래스는 속성과 기능으로 구성되어있다
